# Code of Conduct

Nexum Strategic Archive promotes a professional, academic, and respectful environment.

## Expected Behavior

- Respectful communication
- Evidence-based argumentation
- Intellectual honesty
- Transparency in AI-assisted contributions

## Unacceptable Behavior

- Harassment or discrimination
- Fabrication of data
- Plagiarism
- Commercial misuse of materials

Violations may result in removal of contributions.

This project upholds academic integrity as a non-negotiable standard.
