# Contributing to Nexum Strategic Archive

Thank you for your interest in contributing.

## Principles

All contributions must adhere to:

- Academic rigor
- Transparent methodology
- Source traceability
- Non-commercial usage (CC BY-NC-SA 4.0)

## Types of Contributions

- Analytical reports
- Forensic case studies
- Methodological improvements (5C/2I framework)
- Technical enhancements

## Submission Process

1. Fork the repository
2. Create a dedicated branch
3. Submit a Pull Request with:
   - Clear description
   - Sources cited (APA 7th Edition)
   - Methodological explanation

All submissions are reviewed before integration.

By contributing, you agree that your contribution will be licensed under CC BY-NC-SA 4.0.
